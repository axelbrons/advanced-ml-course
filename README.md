# Advanced Machine Learning
Tous les labs et pre-labs ici

## Contribution
Clone le repo  
```
https://github.com/axelbrons/advanced-ml-pre-lab.git
```

Creer un venv  
```
python -m venv venv  
.\venv\Scripts\Activate.ps1
```

Installe les lib  
```
pip install -r requirements.txt
```

Pas oublier le pull à chaque fois
```
git pull origin main
```
